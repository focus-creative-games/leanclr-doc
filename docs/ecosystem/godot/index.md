# Godot 集成

## 当前状态

LeanCLR 的 Godot 扩展**正在开发中**，预计在 **2026 年 10 月** 发布预览版本。

官方仓库：[focus-creative-games/leanclr-godot](https://github.com/focus-creative-games/leanclr-godot)

:::note
该仓库目前已创建，插件代码尚未公开。请关注仓库更新或社区频道获取预览通知。
:::

## 目标

与 [leanclr-unity](../unity/) 类似，在 Godot 发布管线中集成 LeanCLR 运行时与 LeanAOT，使 GDScript / C# 游戏逻辑能以更小体积与更低内存发布到多平台（含 Web）。
