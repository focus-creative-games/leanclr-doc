## 参与贡献

- 代码：向 [leanclr](https://github.com/focus-creative-games/leanclr) 提交 Pull Request
- 文档：向 [leanclr-doc](https://github.com/focus-creative-games/leanclr-doc) 提交 PR
- 指南见 [贡献指南](../development/contributing)

## 许可证

LeanCLR 运行时与工具链使用 [MIT License](https://github.com/focus-creative-games/leanclr/blob/main/LICENSE)。

Unity 包 [leanclr-unity](https://github.com/focus-creative-games/leanclr-unity) 同样为 MIT。

## 文档反馈

文档问题可在 [leanclr-doc Issues](https://github.com/focus-creative-games/leanclr-doc/issues) 反馈。

运行时 Bug 请在 [leanclr Issues](https://github.com/focus-creative-games/leanclr/issues) 提交，并注明 Unity 版本 / 目标平台 / 复现步骤。
