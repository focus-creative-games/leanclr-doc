## HybridCLR

[hybridclr](https://github.com/focus-creative-games/hybridclr) — Unity 全平台**原生 C# 热更新**方案，与 LeanCLR（发布期后端替换）互补。

商业版 LeanAOT 支持 HybridCLR **DHE** 相关能力，见 [社区版与商业版](../aot/community-vs-commercial)。
